## 概要

VitePress は Vue と Vite をベースにした静的サイトジェネレーター。VuePress の後継であり、Vite による高速な開発サーバーとビルドが特徴。Vue.js の公式ドキュメントでも使用されている。

## Markdown / MDX サポート

- Markdown ベース（MDX 非対応、Vue SFC を埋め込み可能）
- Vue コンポーネントを Markdown 内で直接利用可能
- フロントマターによる設定
- 自動サイドバー生成

## 主な用途

- ドキュメントサイト
- テクニカルブログ
- コンポーネントライブラリのドキュメント
