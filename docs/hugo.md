## 概要

Hugo は Go 言語で書かれた静的サイトジェネレーター。シングルバイナリで動作し、1万ページ以上のサイトでも数秒でビルド可能。テンプレートエンジンに Go テンプレートを使用し、依存関係ゼロで動作する。

## Markdown / MDX サポート

- Markdown を完全サポート（Goldmark パーサー）
- MDX は非対応（代わりにショートコードでリッチな表現を実現）
- フロントマター（TOML/YAML/JSON）による設定
- 画像処理パイプライン内蔵

## 主な用途

- 大規模ドキュメントサイト
- 企業サイト
- ブログ（中〜大規模）
- 多言語サイト

## セットアップ

```bash
# 新規プロジェクト
hugo new site my-site
cd my-site
git init

# 開発サーバー起動
hugo server -D
```
