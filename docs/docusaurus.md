## 概要

Docusaurus は Meta（Facebook）が開発するオープンソースのドキュメントサイトジェネレーター。React + MDX をベースに、バージョン管理、多言語対応（i18n）、Algolia 検索などのエンタープライズ向け機能を標準搭載する。

## Markdown / MDX サポート

- MDX v2/v3 をネイティブサポート
- アドモニション（`:::note`、`:::tip` など）
- プラグインによるカスタマイズ可能な Markdown パイプライン
- ドキュメント・ブログ・ページの3つのコンテンツタイプ

## 主な用途

- プロダクトドキュメント
- API リファレンス
- プロジェクトのランディングページ
- テクニカルブログ
