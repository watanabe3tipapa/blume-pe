## 概要

Nuxt は Vue.js ベースのフルスタックフレームワーク。ファイルベースルーティング、オートインポート、SSR/SSG/ISR などの多彩なレンダリングモードをサポート。`@nuxt/content` モジュールにより Markdown コンテンツ管理が容易。

## Markdown / MDX サポート

- `@nuxt/content` v3 による Markdown / MDX サポート
- ファイルベースのコンテンツ管理（`content/` ディレクトリ）
- SQLite WASM による高速な全文検索
- Vue コンポーネントの埋め込み
- フロントマターによるメタデータ管理

## 主な用途

- フルスタック Web アプリケーション
- ブログ・CMS
- ECサイト
- ドキュメントサイト（`@nuxt/content`）
